1. Reglas generales compartidas

Estas reglas aplican a todos los temas GSEnvios.

1.1. Paleta de colores base

Color Código Muestra
Azul corporativo #0D3B8E
Azul profundo #071F4A
Amarillo dorado #F2B705
Blanco #FFFFFF
Gris claro #F2F2F2

1.2. Metodología BEM (Obligatorio)

Prefijo gsenvios- para TODAS las clases CSS.

Elemento Ejemplo
Bloque .gsenvios-card
Elemento .gsenvios-card__title
Modificador .gsenvios-card--disabled
Elemento + modificador .gsenvios-card__value--highlight

1.3. Regla del amarillo

  • Uso: Solo valores, bordes de acento, botones primarios, indicadores.
  • Prohibido: Fondo principal. Demasiados elementos simultáneos.
  • Sobre fondo oscuro: ✅ Permitido.
  • Sobre fondo claro: ❌ Prohibido. Usar #0D3B8E o #071F4A.

1.4. Tipografía

Elemento Fuente
Títulos y encabezados Montserrat
Texto general, formularios Poppins
Fallback "Segoe UI", Arial, Helvetica, sans-serif
Excepción Tema Alternativo usa SOLO Montserrat

1.5. Prohibido (todos los temas)

  • Colores fuera de la paleta oficial
  • Rojo, verde, morado
  • Efectos de galaxia, cosmos o partículas
  • Brillo excesivo o elementos decorativos innecesarios
  • Sobrecargar la interfaz
  • Amarillo como color dominante
  • Modificar el logo original

3.13. Compatibilidad Tizen Nuevo

Para pantallas Samsung Tizen (S2, S4, S5):

PROHIBIDO:

var(--cualquier-variable) clamp() backdrop-filter flex-wrap display: table-cell vh/vw querySelectorAll let/const/=> Promises/fetch

PERMITIDO:

rgba() float+clearfix linear-gradient radial-gradient inline-block vertical-align XMLHttpRequest setTimeout

Degradados con prefijos:

background: -webkit-linear-gradient(135deg, #0D3B8E, #071F4A);
background: linear-gradient(135deg, #0D3B8E, #071F4A);

Colores directos, sin variables:

/* ✅ Correcto */
.card { background: linear-gradient(135deg, #0D3B8E, #071F4A); color: #FFFFFF; }

/* ❌ Incorrecto */
.card { background: var(--card-bg); color: var(--text); }
